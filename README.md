Pradakshina is a sacred Hindu prayer app developed by CRUX Technologies, 
built with love and devotion for Hindu devotees worldwide.

This app was created to help devotees:
- Count their sacred Pradakshina rounds around temples
- Save their Sankalpa (sacred prayer wish)
- Remember every temple visit through Sacred History
- Receive Brahma Muhurta and Pradosha reminders

This privacy policy was created to be transparent about what 
data we collect (name, email, temple photos) and how we use it 
(only to provide app features — never sold or shared).
